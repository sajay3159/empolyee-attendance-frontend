# Employee Attendance System Frontend

An intuitive application for managing employee attendance. Admins can track, filter, and manage attendance records, and add new employees. Employees can punch in/out and view their own attendance. Built with a clean, scalable React architecture and role-based access control for a smooth and secure user experience.

Tech Stack

React (Vite)
Redux Toolkit + Thunk
React Router v6
